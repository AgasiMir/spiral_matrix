# Матрица: обход по спирали против часовой стрелки

Этот проект предназначен для получения двумерной матрицы по заданному URL, её обработки и вывода элементов в порядке обхода по спирали против часовой стрелки.

## Функционал

- Асинхронное получение данных матрицы по HTTP-запросу.
- Построение двумерной матрицы из полученных данных.
- Обход элементов матрицы по спирали, начиная с внешнего контура и двигаясь внутрь, против часовой стрелки.
- Вывод плоского списка элементов в порядке обхода.

## Используемые функции

В модуле `get_matrix.py` реализованы следующие основные функции:

- `get_matrix(url: str) -> list[int]` — асинхронно получает данные по URL, строит матрицу и возвращает элементы, обойдённые по спирали.
- `make_matrix(data: list[list[int]]) -> list[list[int]]` — создаёт двумерную матрицу из входных данных.
- `spiral_counter_clockwise(matrix: list[list[int]]) -> list[int]` — выполняет обход матрицы по спирали против часовой стрелки и возвращает список значений.

## Тестирование

Проект включает автоматические тесты, написанные с использованием pytest.
Для запуска тестов выполните команду: `pytest`

Для подробного вывода (с отображением имён тестов) используйте флаг -v: `pytest -v`

Тесты проверяют:
- Корректность обхода матрицы по спирали.
- Правильность построения матрицы.
- Корректность асинхронного получения данных.
